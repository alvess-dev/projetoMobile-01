## Configuração do projeto
**1. Clonando o Repositório**
```bash
  git clone https://github.com/alvess-dev/projetoMobile-01.git
  cd projetoMobile-01
```

Rode o seguinte comando no diretório do projeto:
  ```bash
  npm install && npm install -g @ionic/cli
```
